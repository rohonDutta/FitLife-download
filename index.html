From step tracking to AI insights — FitLife covers every corner of your fitness journey.

🚶

Step Tracking

Real-time hardware step counter with calorie burn, distance, and active minutes. 30-day history, streaks, and manual entry.

💪

Workout Manager

Build custom workouts or use pre-built templates. Strength, Cardio, HIIT, Yoga, and more with auto-sync to Health Connect.

🔥

Calorie & Macro Tracking

Log meals by type with 100+ built-in foods plus live search via Open Food Facts API. Real-time macro dashboard.

Get started

Install in 4 easy steps

01

Download the APK

Tap the download button and save the FitLife-release.apk file to your Android device.

02

Allow Unknown Sources

Go to Settings → Apps → Special app access → Install unknown apps and enable it for your file manager or browser.

03

Open & Install

Tap the downloaded APK file and follow the on-screen prompts to complete installation. It only takes a few seconds.

04

Grant Permissions

On first launch, grant Activity Recognition and Health Connect access. Complete the onboarding wizard and you're ready to go!
